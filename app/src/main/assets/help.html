Passwort Manager

Start der App:

ausschließlich mit einem Fingerabdruck oder der Geräte PIN.

Neue Einträge:

mit dem großen "+" - Zeichen rechts unten auf dem Bildschirm; alternativ über das Menü "hinzufügen".
Alle Felder müssen mit mindestens einem Zeichen gefüllt sein.

Suche nach einem Eintrag: 

Das Lupensymbol drücken und ein paar Buchstaben eingeben, durchsucht wird das Feld "Name des Eintrags".

Datensatz ändern:

Klicken Sie auf den gewünschten Eintrag und ändern die Daten. Speichern Sie anschließend die Änderungen mit einem klick auf "Eintrag ändern" ab.

Datensatz löschen:

Klicken Sie auf den gewünschten Eintrag und auf der folgenden Seite drücken Sie auf "Eintrag löschen". Der Eintrag wird unwiederbringlich gelöscht.

Passwort in die Zwischenablage kopieren:

Drücken Sie für ca. eine Sekunde auf den Eintrag und das Passwort wird in die Zwischenablage kopiert. Sofern vorhanden vibriert das Handy kurz und es erscheint eine Mitteilung auf dem Bildschirm ("Passwort kopiert).

Datei importieren:

Neue Einträge können auch über eine Datei hinzugefügt werden, die mit dem (Browser) Programm "PWMDatenbank.html" erzeugt wurde.
Diese Datei wird z.B. per Mail an das Smartphone übertragen und eingelesen. Hierzu geben Sie dasselbe Passwort ein, das Sie auch im o.g. Programm für den Export verwendet haben.
Sie wählen die Importdatei mit dem Button "Auswahl der Importdatei" aus und drücken dann den Button "importiere die Datenbank". Anschließend erscheinen die importierten Einträge in der Liste.

Datenbank löschen:

Nach einer Sicherheitsabfrage wird die gesamte Datenbank gelöscht, eine Wiederherstellung ist nicht möglich.

Programm beenden:

Auf das "Schloß-Symbol" klicken und die App wird beendet.

Versionsstand:

Die Version ist 1.0.17.
